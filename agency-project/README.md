# Agency Testing Project

This is a test React project for the Agency-Testing repository.

## Project Structure

- `/components` - Reusable React components
- `/guidelines` - Design guidelines and documentation
- `/styles` - CSS and styling files
- `/supabase/functions/server` - Supabase functions
- `/utils/supabase` - Supabase utilities

## Setup

To get started with this project:

1. Clone the repository
2. Install dependencies with `npm install`
3. Configure environment variables
4. Run `npm start` to start the development server

## Technologies Used

- React
- Supabase
- TypeScript
- Tailwind CSS

## Contributing

Please follow the guidelines in the `/guidelines` folder when contributing to this project.
